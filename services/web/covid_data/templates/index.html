A Flask full of Pandas

Simple web interface using Flask and Pandas to display data from U.K. Office of National Statistics (ONS) - in this case Deaths registrations

The links in the left menu are the sheets in the Excel file, these lead to graphical displays of time series and weekly grouped data using Googles Charts Visualisation package.

Published in the hope the example code is useful to someone and as an aide de memoire for me.
